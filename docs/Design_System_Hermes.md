# Diretrizes de Identidade Visual - Sistema Hermes

Este documento estabelece as diretrizes de identidade visual para o desenvolvimento do Sistema Hermes, garantindo a conformidade com o **Manual de Marca do Governo do Estado do Rio de Janeiro** (Outubro 2023) [1].

## 1. Cores Oficiais

A paleta de cores primária e de apoio deve ser utilizada para elementos de interface, gráficos e destaques, conforme as especificações abaixo:

| Nome da Cor | HEX | RGB | Pantone | Uso Sugerido |
| :--- | :--- | :--- | :--- | :--- |
| **Azul (Primário)** | `#005A92` | 0 / 90 / 146 | 3015 C | Cabeçalhos, botões primários, links. |
| **Cinza (Apoio)** | `#A0A0A0` | 160 / 160 / 160 | Cool Gray 7 C | Textos de apoio, bordas, elementos de interface inativos. |
| **Dourado (Destaque)** | `#C99D32` | 201 / 157 / 50 | 117 C | Destaques, ícones, elementos gráficos. |
| **Preto (Texto)** | `#000000` | 0 / 0 / 0 | Black C | Textos principais. |
| **Verde (Apoio)** | `#427842` | 66 / 120 / 66 | 3415 C | Elementos de feedback positivo (sucesso). |

## 2. Tipografia

A tipografia é crucial para a legibilidade e a consistência da marca.

| Uso | Fonte Oficial | Fonte de Apoio (Web Safe/Open Source) | Estilos |
| :--- | :--- | :--- | :--- |
| **Títulos e Destaques** | Antenna | **Roboto** | Bold, Black |
| **Corpo de Texto** | Antenna | **Roboto** | Regular, Light |

**Recomendação:** Para o desenvolvimento web, a fonte **Roboto** é a fonte de apoio oficial e deve ser priorizada, pois é *open source* e garante excelente renderização em navegadores e dispositivos móveis.

## 3. Logotipo e Aplicação

O sistema Hermes deve utilizar o logotipo oficial do Detran-RJ, respeitando as regras de área de respiro e tamanho mínimo estabelecidas no manual.

*   **Logotipo:** Deve ser posicionado no canto superior esquerdo do cabeçalho do sistema.
*   **Versões:** Priorizar a versão colorida em fundos claros e a versão negativa (branca) em fundos escuros (como o azul primário).
